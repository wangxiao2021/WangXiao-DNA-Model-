# DNA 数字孪生结构模型

一个交互式的高中生物课本级 DNA 分子结构可视化工具。

## 功能特性

- **教科书标准结构** — 按照高中生物教科书的标准呈现 DNA 双螺旋结构
- **完整的分子成分** — 包括磷酸基团、脱氧核糖（五碳糖）、碱基（A/T/C/G）、磷酸二酯键和氢键
- **反向平行展示** — 两条 DNA 链以反向平行方式排列，符合生物学规律
- **交互式体验** — 支持鼠标拖动旋转、滚轮缩放、点击节点查看详细信息
- **低调高级风格** — 钢琴黑背景、柔和光照、无塑料感的材质表现

## 使用方法

### 本地运行

1. 克隆或下载本项目
2. 使用 VS Code 的 Live Server 插件打开 `index.html`
3. 或使用 Python 本地服务器：
   ```bash
   python -m http.server 8000
   ```
   然后访问 `http://localhost:8000`

### 交互操作

- **旋转** — 鼠标拖动
- **缩放** — 滚轮滚动
- **自动旋转** — 不操作时自动旋转（拖动时停止）
- **查看详细信息** — 点击任意碱基、糖环或键

## 结构说明

### 磷酸基团 (P)
- **颜色** — 紫色
- **形状** — 球体
- **位置** — 最外侧
- **描述** — 连接 DNA 主链的关键组成

### 脱氧核糖（五碳糖）
- **颜色** — 灰色
- **形状** — 正五边形柱体
- **位置** — 中间
- **描述** — DNA 骨架的糖部分

### 含氮碱基
- **腺嘌呤 (A)** — 绿色，凸起薄片，与 T 配对
- **胸腺嘧啶 (T)** — 橙红色，凹陷薄片，与 A 配对
- **胞嘧啶 (C)** — 金黄色，凹陷薄片，与 G 配对
- **鸟嘌呤 (G)** — 青色，凸起薄片，与 C 配对

### 连接
- **磷酸二酯键** — 灰蓝色实线，连接相邻核苷酸
- **氢键** — 蓝色虚线，连接互补碱基对

## 技术栈

- **Three.js** — 3D 图形渲染
- **OrbitControls** — 相机交互控制
- **ES6 模块化** — 现代 JavaScript 语法
